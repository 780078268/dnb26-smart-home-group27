# DnB26 Smart Home - Group 27 Software

第 27 组智能家居原型软件组资料仓库。

本仓库用于管理软件组阶段一和阶段二的公开资料、接口契约、YOLO 环境说明与项目协作文档。软件组目标是把 Orange Pi / 华为开发板、摄像头、传感器和执行机构接入到可演示的信息系统中，完成状态展示、历史查询、远程控制、YOLO 目标识别和人脸识别门禁。

## 文件说明

- `第27组软件组分工与接口文档.md`：五人分工、开发计划、系统结构、软硬件接口和验收 checklist。
- `第27组软件接口契约.openapi.yaml`：前端、后端、板端网关之间的 OpenAPI 接口契约。
- `DnB26题目-智能家居.docx`：课程项目题目。
- `DnB26_Introduction.pptx`：课程介绍和交付要求。
- `Yolo安装与环境配置.pdf`：YOLO 环境配置资料。
- `yolo训练方法.docx`：YOLO 训练资料。

## 隐私说明

本 public 仓库不包含分组名单表。`分组名单.xlsx`、临时文件、运行数据库、上传图片和本地环境变量已通过 `.gitignore` 排除。

## 推荐开发边界

- 板端/网关：Orange Pi / 华为开发板，优先通过 HTTP 接入后端；无法直接联网时使用串口转 HTTP 网关。
- 后端：提供 REST API、SQLite 数据库、命令队列和图片保存。
- 前端：提供状态仪表盘、历史查询、远程控制和识别结果展示。
- 识别：YOLO 负责目标识别，人脸识别负责 2 真 1 假门禁演示。
