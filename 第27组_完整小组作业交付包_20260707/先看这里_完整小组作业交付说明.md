## 2. 如何运行完整项目

在 Windows PowerShell 中：

```powershell
cd 本文件夹路径
python -m venv .venv
.\.venv\Scripts\python.exe -m pip install --upgrade pip
.\.venv\Scripts\python.exe -m pip install -r requirements.txt
$env:HOST="0.0.0.0"
.\.venv\Scripts\python.exe backend\server.py
```

浏览器打开：

```text
http://127.0.0.1:8000
```

局域网内 Orange Pi 访问：

```text
http://电脑局域网IP:8000
```

## 3. 推荐演示顺序

1. 打开 GUI。
2. Step 1：点击“使用真实无人机样本”，运行 YOLO 识别。
3. Step 1：点击“使用真实灭火器样本”，运行 YOLO 识别。
4. Step 1：点击“使用真实灯泡样本”，运行 YOLO 识别，观察 `SET_LIGHT level=80`。
5. Step 2：点击 `真 P01`、`真 P02`、`假 U01`，完成 2 真 1 假。
6. Step 4：按顺序点击状态上报、下发命令、设备确认执行。
7. Step 5：查看历史记录和事件流。

## 5. Orange Pi 对接

如果只给 Orange Pi 同学对接，请看：

```text
docs/OrangePi部署说明.md
scripts/orange_pi_device_client.py
```

Orange Pi 端运行示例：

```bash
python3 scripts/orange_pi_device_client.py --server http://电脑局域网IP:8000 --loop
```

硬件同学主要改：

```python
execute_command(command)
```

把打印语句替换成 GPIO、PWM、继电器、舵机等真实控制。

## 7. 注意

- 不包含 `.venv`，接收方需要按 `requirements.txt` 安装依赖。
- 不包含原始大数据集 `lfw_raw`、`open_images_raw`、训练缓存 `runs`，避免交付包过大。
- 完整 YOLOv8m 在 Orange Pi CPU 上可能很慢，推荐电脑跑识别，Orange Pi 做硬件端。
