```text

```

## 5. 仍需硬件同学完成的部分

软件侧已经完成接口和演示闭环，硬件侧还需完成：

- Orange Pi 读取真实温度、门窗、灯光状态。
- Orange Pi 控制真实灯光、风扇、门禁执行器。
- 如使用摄像头，Orange Pi 调用摄像头拍照并上传到 `/api/device/photos`。
- 将 `scripts/orange_pi_device_client.py` 中的 `execute_command()` 从打印改为 GPIO/PWM/继电器/舵机控制。
